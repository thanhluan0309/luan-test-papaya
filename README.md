# Pumpkin — Software Artisan Challenges

This repository contains my solutions to two challenge tracks.

---

## [Result_AI_Engineering_Challenges/](./Result_AI_Engineering_Challenges/)

**Live demo:** [https://luan-papaya-chanllenge.vercel.app](https://luan-papaya-chanllenge.vercel.app)

15 hands-on AI engineering challenges in the insurance domain, built with Next.js 16, React 19, and TypeScript. Topics range from data cleaning and glossary search to fraud detection, LLM-powered agents, state machines, SDKs, and multi-tenant config platforms.

| Track       | Challenges | Stack                            |
| ----------- | ---------- | -------------------------------- |
| Beginner    | 01–05      | React, TypeScript, Tailwind      |
| Intermediate| 06–09      | Next.js API routes, Recharts     |
| Advanced    | 10–15      | LLM, state machines, Vitest, SDK |
